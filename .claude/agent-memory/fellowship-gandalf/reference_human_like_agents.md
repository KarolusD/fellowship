Specific guide: https://claudefa.st/blog/guide/agents/human-like-agents

Guide on making Claude Code agents feel like senior developers rather than task executors. Key patterns: reasoning transparency (think aloud before solving), uncertainty as expertise signal ("I think..." > false confidence), role-based personality switching per task type, natural conversation over command-style prompts.

**How to apply:** Reference when refining companion personalities, especially for new companions (Boromir, Aragorn, Merry, Sam, Arwen, Bilbo). Validates the Fellowship's existing approach — we're already doing personality injection and role-based switching. Worth checking for patterns we haven't adopted yet.
