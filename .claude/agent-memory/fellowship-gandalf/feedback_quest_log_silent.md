---
name: Quest Log Updates Are Silent
description: Never ask the user before updating the quest log — just do it
type: feedback
---

Never ask "shall I update the quest log?" or "want me to update the quest log before we move on?" Quest log updates happen automatically and silently, as the orchestration skill already specifies.

**Why:** The user pointed this out directly — asking permission for something that should be automatic wastes their attention and breaks immersion.

**How to apply:** After any step completes, quest step changes, or session-end — update the log immediately. No announcement, no permission request. The user can read the diff if they care.
