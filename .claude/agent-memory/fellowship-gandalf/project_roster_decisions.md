Roster decisions confirmed 2026-03-23 after brainstorming session and reference implementation research.

**Why:** Initial roster had Sam as User Researcher and Bilbo as UX Writer. These were reconsidered for lore accuracy and cleaner role separation.

**Decisions:**

- **Sam** → DevOps / Infrastructure (was: User Researcher)
  - Lore: gardener → tends infrastructure; supply lines → pipelines and deployments
  - Tools: Read, Glob, Grep, Bash, WebSearch, WebFetch (Bash added for infra commands)
  - Loads: devops/SKILL.md (new skill to be built)

- **Bilbo** → Technical Writer / Documentation (was: UX Writer)
  - Lore: author of There and Back Again + Translations from the Elvish; the map makes the quest repeatable
  - Loads: documentation/SKILL.md (new skill to be built)

- **Arwen** → Product Designer + UX Writing (UX writing absorbed from Bilbo)
  - Now works in three modes: Figma artifacts, UX/a11y audits, and microcopy (empty states, errors, onboarding)
  - Loads: design/SKILL.md + ux-writing/SKILL.md

- **Aragorn** → Product Manager + Tier 4 Scope Guardian (expanded role)
  - Before any Tier 4 work begins, Gandalf consults Aragorn on scope
  - Prevents unnecessary parallel work — the king who walked the Paths of the Dead knows which gambles are necessary

**How to apply:** When building new companion specs, refer to these decisions. When dispatching Aragorn, include Tier 4 scope consultation in his responsibilities.
