# Fellowship Learnings

Observations discovered during execution — things that aren't part of any plan but would be valuable in future sessions. Append-only, date-grouped.

**What belongs here:**
- Library/API quirks not obvious from docs
- Codebase constraints discovered the hard way
- Tooling gotchas (test commands, build flags, config formats)
- Process observations (what tier worked, what didn't)
- Environment-specific issues (platform behavior, service limits)

**What doesn't belong here:**
- Design decisions (those go in specs)
- User preferences (Claude's auto-memory handles those)
- Task status (that's the quest log)

**Format:** `- [category] observation text`

Categories: `engineering`, `tooling`, `codebase`, `process`, `environment`
